Detailed Description

file containing functions to run the environment for the user

Author
Luciano Mantovani
Date
October 2014 this file has functions that react to the lines entered by the user. The commands from the file environmentcommands.c are called by functions from this file.

this function parses an entry in the environment

it parses the entire line entered and stores the command in the field char *command from the struct mipsstr, and stores all the arguments in the field char *argenv[] from the structure mipsstr

int runenv ( struct mipsstr *  mips)

this function runs the environment program and allows the user to enter commands

after obtaining a line from stdin, it parses it and analizes the command. The function analize() also executes it, and if there's no report (mips->report = 0) the program leaves. F
