struct ptype* parseline ( struct ptype *  data)

this function parses the line entered by user and identifies the tag, the operation, the arguments, and gets rid of the commentaries

Parameters
*dataA pointer to a ptype structure, created by the main program.
Returns
it returns the pointer to the same structure, but it modifies data->operation and data->argline[]
Note
this functions is meant to parse lines from an elf script, not the commands from the enviroment. there's another file with functions for that
