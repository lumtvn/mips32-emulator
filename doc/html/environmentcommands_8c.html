Detailed Description

file containing functions that execute the environment commands

Author
Luciano Mantovani
Date
October 2014 this file has all the functions that are called by the function analize() in file environment.c they execute each and every possible combination of the command and its arguments

to retrieve the register passed by the argument which holds the register name in string format, we use a set of functions taken from the book "C Programming" by Dennis Ritchie and Brian Kernighan. This functions are hash(), lookup(), and install(). They allow the creation of a hash table.

See Also
for more information about lookup: http://zanasi.chem.unisa.it/download/C.pdf
