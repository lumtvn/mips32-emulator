Detailed Description

file containing functions meant to assemble the code from an elf script

Author
Luciano Mantovani
Date
October 2014 This file has functions called by the main program when it has to assemble an elf script.

the only parameter taken in this and in every function in all source files is a pointer to a struct data type. It makes it easier to handle information between functions.
